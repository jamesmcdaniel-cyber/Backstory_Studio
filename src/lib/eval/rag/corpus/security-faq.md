# Backstory Sales AI — Security FAQ

- Customer data is encrypted at rest and in transit.
- Backstory is SOC 2 Type II compliant; the audit report is available under NDA.
- Per-org webhook signing secrets are provided on the Enterprise tier so customers can verify inbound webhook authenticity.
- Data residency is US-only today; an EU residency option is planned for the second half of the year (H2).
- Customer data is never used to train Backstory's underlying models — each organization's data stays isolated to that organization.
