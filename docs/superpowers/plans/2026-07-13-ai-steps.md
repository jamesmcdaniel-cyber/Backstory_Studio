# AI Steps Suite Implementation Plan (WS14)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Spec `docs/superpowers/specs/2026-07-13-ai-steps-design.md` — a first-class `ai` step (ask/extract/categorize/summarize/score) runnable without a roster agent, with structured outputs, full builder + copilot ripple.

**Architecture:** One zod node type with an `aiOp` discriminator (the `data`/`dataOp` house pattern). Interpreter routes it through the existing `RunActionFn` adapter seam (kind gains `'ai'`) inside the same retry/timeout policy as tool/http. The execute-flow adapter builds prompts via pure builders, calls `createModelRunner`, and reuses `structuredResponseInstruction`/`parseStructuredAgentOutput` for structured ops.

**Tech Stack:** zod graph schema, pure interpreter (node:test), model-runner (`src/lib/llm/model-runner.ts`), jsdom component tests.

```ts
// graph.ts
export const AI_OPS = ['ask', 'extract', 'categorize', 'summarize', 'score'] as const
export type AiOp = (typeof AI_OPS)[number]
export const AI_OP_LABELS: Record<AiOp, string> = {
  ask: 'Ask AI', extract: 'Extract data', categorize: 'Categorize', summarize: 'Summarize', score: 'Score',
}
const aiNode = z.object({
  id: z.string(),
  type: z.literal('ai'),
  data: z.object({
    aiOp: z.enum(AI_OPS),
    input: z.string().optional(),
    instructions: z.string().optional(),
    model: z.enum(['fast', 'smart']).optional(),
    outputFields: z.array(outputFieldSchema).optional(),
    categories: z.array(z.string()).optional(),
    scoreMin: z.number().optional(),
    scoreMax: z.number().optional(),
    label: z.string().optional(),
    note: z.string().optional(),
    onError: z.enum(['stop', 'continue', 'route']).optional(),
    retries: z.number().int().min(0).max(5).optional(),
    timeoutMs: z.number().int().min(1000).max(AGENT_RUN_TIMEOUT_MS).optional(),
  }),
})
```
